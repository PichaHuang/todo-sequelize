# 介紹  
簡易的代辦事項, 資料庫改成關聯式 MySQL  

# 功能  
登入後可新增瀏覽修改刪除代辦事項

# 使用  
安裝node.js and npm  
clone 專案  
npm install  
npm run dev  
瀏覽器輸入 http://localhost:3000  
npm run seed
